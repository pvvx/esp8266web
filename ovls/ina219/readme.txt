ina219.ovl

Драйвер для работы с датчиком INA219

Инициализация:
mdb[60] - Номер пина (0..15) для сигнала SCL, по умолчанию GPIO4
mdb[61] - Номер пина (0..15) для сигнала SDA, по умолчанию GPIO5
Переменные:
mdb[62] - Счетчик опроса датчика. При смене указывает, что с датчика обработаны новые данные. 
mdb[63] - Флаг: 
			=0 - данные верны, ошибок нет. 
			=-1 - драйвер закрыт.
			=-2 - неисправность сенсора
			=-3 - ошибки в параметрах инициализации (номера пинов SCL, SDA)
mdb[64] - напряжение (1 мВ)
mdb[65] - ток (50 мкА)
mdb[66] - мощность (1 мВт)
mdb[67] - сенсор тока
